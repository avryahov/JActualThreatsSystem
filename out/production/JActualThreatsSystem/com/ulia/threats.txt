По видам возможных источников УБПДн;угрозы, связанные с преднамеренными или непреднамеренными действиями лиц, имеющих доступ к ИСПДн, включая пользователей ИСПДн, реализующих угрозы непосредственно в ИСПДн (внутренний нарушитель);
По видам возможных источников УБПДн;угрозы, связанные с преднамеренными или непреднамеренными действиями лиц, не имеющих доступа к ИСПДн, реализующих угрозы из внешних сетей связи общего пользования и (или) сетей международного информационного обмена (внешний нарушитель)
По типу ИСПДн, на которые направлена реализация УБПДн;угрозы безопасности ПДн, обрабатываемых в ИСПДн на базе автономного автоматизированного рабочего места (АРМ);
По типу ИСПДн, на которые направлена реализация УБПДн;угрозы безопасности ПДн, обрабатываемых в ИСПДн на базе АРМ, подключенного к сети общего пользования (к сети международного информационного обмена);
По типу ИСПДн, на которые направлена реализация УБПДн;угрозы безопасности ПДн, обрабатываемых в ИСПДн на базе локальных информационных систем без подключения к сети общего пользования (к сети международного информационного обмена);
По типу ИСПДн, на которые направлена реализация УБПДн;угрозы безопасности ПДн, обрабатываемых в ИСПДн на базе локальных информационных систем с подключением к сети общего пользования (к сети международного информационного обмена);
По типу ИСПДн, на которые направлена реализация УБПДн;угрозы безопасности ПДн, обрабатываемых в ИСПДн на базе распределенных информационных систем без подключения к сети общего пользования (к сети международного информационного обмена);
По типу ИСПДн, на которые направлена реализация УБПДн;угрозы безопасности ПДн, обрабатываемых в ИСПДн на базе распределенных информационных систем с подключением к сети общего пользования (к сети международного информационного обмена)
По способам реализации УБПДн;угрозы, связанные с НСД к ПДн (в том числе угрозы внедрения вредоносных программ);
По способам реализации УБПДн;угрозы утечки ПДн по техническим каналам утечки информации;
По способам реализации УБПДн;угрозы специальных воздействий на ИСПДн.
По виду несанкционированных действий, осуществляемых с ПДн;угрозы, приводящие к нарушению конфиденциальности ПДн (копированию или несанкционированному распространению), при реализации которых не осуществляется непосредственного воздействия на содержание информации;
По виду несанкционированных действий, осуществляемых с ПДн;угрозы, приводящие к несанкционированному, в том числе случайному, воздействию на содержание информации, в результате которого осуществляется изменение ПДн или их уничтожение;
По виду несанкционированных действий, осуществляемых с ПДн;угрозы, приводящие к несанкционированному, в том числе случайному, воздействию на программные или программно-аппаратные элементы ИСПДн, в результате которого осуществляется блокирование ПДн.
По используемой уязвимости;угрозы, реализуемые с использованием уязвимости системного ПО;
По используемой уязвимости;угрозы, реализуемые с использованием уязвимости прикладного ПО;
По используемой уязвимости;угрозы, возникающие в результате использования уязвимости, вызванной наличием в АС аппаратной закладки;
По используемой уязвимости;угрозы, реализуемые с использованием уязвимостей протоколов сетевого взаимодействия и каналов передачи данных;
По используемой уязвимости;угрозы, возникающие в результате использования уязвимости, вызванной недостатками организации ТЗИ от НСД;
По используемой уязвимости;угрозы, реализуемые с использованием уязвимостей, обусловливающих наличие технических каналов утечки информации;
По используемой уязвимости;угрозы, реализуемые с использованием уязвимостей СЗИ.
По объекту воздействия;угрозы безопасности ПДн, обрабатываемых на АРМ;
По объекту воздействия;угрозы безопасности ПДн, обрабатываемых в выделенных средствах обработки (принтерах, плоттерах, графопостроителях, вынесенных мониторах, видеопроекторах, средствах звуковоспроизведения и т.п.);
По объекту воздействия;угрозы безопасности ПДн, передаваемых по сетям связи;
По объекту воздействия;угрозы прикладным программам, с помощью которых обрабатываются ПДн;
По объекту воздействия;угрозы системному ПО, обеспечивающему функционирование ИСПДн.
